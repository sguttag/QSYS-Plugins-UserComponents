This Q-SYS plugin is for working with Dolby DSS type cinema servers (DSS100, DSS200, DSS220).  It has been tested on DSS200 and DSS220 servers running 4.9.5.2 using QDS 9.13.1 LTS.

For more in-depth information, please download the help file, which contains both a pdf and htm versions. 

Below are screenshots of the pages.  Note, the audio processor and CAT745 Mediablock must be configured in properties for their respective controls to show.
